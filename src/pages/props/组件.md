# Props 是什么（本质）
Props = 父组件传给子组件的数据（只读）
```
组件 = 函数
Props = 函数参数
```

# Props 的核心特性
### 单向数据流
```
父组件 → 子组件（只能单向）
```

### Props 是只读的（immutable）
Props = 输入参数，不允许修改

### 组件通信
**父组件通过 props的普通参数，诸如普通类型、对象来实现父组件传递到子组件；然后通过传递函数参数来实现子组件通知父组件**

# Props.children
children属性代表的是组件的所有子dom节点

```
<Card>
  <h1>标题</h1>
  <p>内容</p>
</Card>
```
实际传递的是  
```
<Card children={<h1>标题</h1>} />
```
